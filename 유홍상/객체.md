# 객체(Object)

## 1. 객체(Object)란?
- 자바스크립트의 핵심 자료 구조  
  - 원시 타입(숫자, 문자열, 불리언, null, undefined, symbol, bigint)을 제외한
    나머지 값(배열, 함수, 정규표현식 등)은 모두 객체이다.
- 객체는 데이터와 기능을 함께 묶는 컨테이너  
  - **프로퍼티**: 키(key)와 값(value)의 쌍으로 이루어진 데이터  
  - **메서드**: 프로퍼티 값이 함수인 경우

```javascript
const user = {
  name : "홍길동",
  age : 27,
  introduce : function() {
    console.log(`${this.name}은 ${this.age}살 입니다.`);
  }
};

user.introduce();
```
### 객체의 특징
- 동적: 실행 중에도 프로퍼티를 추가, 삭제 가능
- 참조 타입: 같은 객체를 여러 변수에서 참조 가능
- 중첩 객체: 객체 안에 객체 포함 가능

## 2. 객체 생성 방법
**자바스크립트에서는 다양한 방식으로 객체를 만들 수 있습니다.**

### 1. 객체 리터럴
- 중괄호 {} 안에 key: value 형식으로 작성
- 가장 간단하고 직관적인 방법(가장 많이 사용됨.)

```javascript
const person {
  name: "홍길동",
  age: 20,
  gender: "male"
};
console.log(person.name) // 홍길동
console.log(person["age"]) // 20
```

### 2. Object 생성자 함수
- new Object()로 빈 객체를 만든 뒤 프로퍼티를 추가하는 방식
- 잘 쓰이지 않음 (리터럴 방식이 더 편리함)

```javascript
const person = new Object();
person.name = "홍길동";
person.age = 20;

console.log(person); // { name : "홍길동", age : 20};
```

### 3. 생성자 함수
- new 키워드와 함께 호출되는 함수로 객체를 찍어내듯 생성
- 비슷한 객체를 여러 개 만들 때 유용
- 관례상 대문자로 시작

```javascript
function Person(name, age) {
  this.name = name;
  this.age = age;
  this.introduce = function() {
    console.log(`${this.name}은 ${this.age}살 입니다.`);
  }
}
const person1 = new Person("홍길동", 20);
person1.introduce(); // 홍길동은 20살 입니다.
```

## 프로퍼티
### 키(key) 규칙
- 문자열 또는 Symbol만 가능 (숫자 -> 내부적으로 문자열로 변환됨)
- 유효한 식별자일 경우 따음표 생략 가능
- 식별자 규칙:
  - 문자, 밑줄(_), 달러($)로 시작 가능
  - 숫자로 시작 불가
  - 공백, 특수문자 포함 불가 (이 경우 반드시 따음표 필요)
```javascript
const obj = {
  name : "길동",  // 식별자 가능
  "first-name" : "홍",  // 따음표 필수
  1 : "one" // 내부적으로 "1"로 처리됨
};
// obj.first-name X 에러
```

## 프로퍼티 접근
### 1. 마침표 표기법
```javascript
// 마침표 표기법에 의한 프로퍼티 접근
var person {
  name : 'Lee'
};

console.log(person.name); // Lee
```
### 2. 대괄호 표기법 -> 따음표로 감싼 문자열, 동적 표현식 가능
```javascript
// 대괄호 표기법에 의한 프로퍼티 접근
var person {
  name : 'Lee'
};

console.log(person['name']); // Lee
```

## 프로퍼티 추가/수정/삭제
```javascript
const user = { name : "홍길동", age : 27 };
// 동적 추가
user.hobby = "코딩";

// 수정
user.age = 30;

// 삭제
delete user.name

console.log(user) // { age : 30, hobby : "코딩" }
```

## 메서드
* 프로퍼티 값이 함수일 경우 일반 함수와 구분하기 위해 메서드라 부른다.
-> 즉, 메서드는 객체에 묶여 있는 함수를 의미한다.
* this를 통해 자신을 참조
```javascript
const circle = {
  redius = 5, // <- 프로퍼티

  // 원의 지름
  diameter : function () {
    return 2 * this.redius; // this는 circle 객체를 가리키는 참조변수다.
  }
};

console.log(circle.diameter()); // 10
```

## 객체 순회
### for...in(키 순회)
```javascript
const user = { id : 1, name : "홍상"};

for (let key in user) {
  console.log(key, user[key]);
}
// id 1
// name 홍상
```
⚠️ 배열에는 사용하지 않는 게 좋음 (예상치 못한 순서 + 추가 프로퍼티까지 순회)

### for...of(값 순회, 이터러블 전용)
```javascript
const arr = ["a", "b", "c"];

for (let value of arr) {
  console.log(value);
}
// a
// b
// c
```
➡️ 객체에는 바로 사용 불가 (TypeError: obj is not iterable)

## ES6 객체 리터럴 확장 기능
ES6에서는 더 간결하고 표현력 있는 객체 리터럴의 확장 기능을 제공한다.(아래 3가지)
  1. 프로퍼티 축약 표현
  2. 계산된 프로퍼티 이름
  3. 메서드 축약 표현

### (1) 프로퍼티 축약 표현
#### 개념
**변수명과 프로퍼티 키가 같은 이름일 때 키: 값을 생략해 짧게 쓸 수 있습니다.**
```javascript
// ES5 방식
let x = 10;
let y = 20;

const obj = {
  x : x,
  y : y
} 
console.log(obj); // { x : 10, y : 20 }

// ES6 방식
let x = 10;
let y = 20;

const point2 = { x, y };
console.log(obj); // { x : 10, y : 20 }
```
**자주 하는 실수**
- “축약하면서 이름도 바꾸기”는 불가 → 축약은 **‘동일’**할 때만.

### (2) 계산된 프로퍼티 이름
#### 개념
**프로퍼티 키를 표현식으로 계산해 만들 수 있습니다. 대괄호 [] 안에 임의의 표현식 넣기**
```javascript
const key = 'score';
const idx = 1;
const sym = Symbol('id');

const user = {
  ['first' + 'Name']: 'Lee', // "firstName: Lee
  [key]: 95,                 // "score": 95
  ['item' + idx]: 'book',    // "item1": "book"
  [sym]: 123                 // 심벌 키
};

user.firstName; // "Lee"
user['item1'];  // "book"
user[sym];      // 123
```

### (3) 메서드 축약 표현
#### 개념
**객체의 함수 프로퍼티를 정의할 때 function 키워드와 :를 생략해 짧게 쓸 수 있습니다.**
```javascript
  // // ES5 방식
  let obj = {
  name = 'Lee'
  sayHi: function () {
    console.log('Hi!' + this.name);
  }
};

obj.sayHi(); // Hi! Lee

// ES6 방식
const obj = {
  name = 'Lee'
  // 메서드 축약 표현
  sayHi() {
    console.log('Hi!' + this.name);
  }
};

obj.sayHi(); // Hi! Lee
```

## 핵심 정리
- 객체는 데이터와 기능을 묶는 구조 (프로퍼티 + 메서드)
- 객체 생성 방법 -> 리터럴 / Object 생성자 / 생성자 함수
- 프로퍼티 접근 방식 -> 점 표기법(.), 대괄호 표기법([])
- 없는 프로퍼티 접근 시 -> undefined 반환
- 프로퍼티 추가/변경/삭제 가능 (delete)
- 순회 방식:
  - for...in: 객체 키 순회
  - for...of: 배열, 문자열 값 순회
- ES6 객체 리터럴 확장 기능:
  - 변수명과 키가 같다면 무조건 축약으로 가독성 ↑
  - 객체의 동작은 메서드 축약으로 표현
  - 동적으로 키가 필요하면 계산된 프로퍼티 이름 사용
